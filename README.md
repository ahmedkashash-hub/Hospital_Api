# Hospital Management API

## Overview
A backend system for managing hospital operations including patients, doctors, and appointments.

## Tech Stack
- ASP.NET Core Web API
- SQL Server

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Authentication & Authorization
- Validation & Error Handling

## Architecture
- Clean Architecture / Layered Design

## How to Run
1. Clone repo
2. Setup database
3. Run project
